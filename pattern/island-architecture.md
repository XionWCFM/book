# island architecture

아일랜드 아키텍처를 이용하면 서버렌더링 되는 웹 앱 내에서 작은 단위의 인터랙트에 집중한 분리된 코드 조각을 만들게 됩니다.

아일랜드 아키텍처는 점진적으로 향상되는 html을 만들어낼 수 있습니다. 

아일랜드의 상호작용을 위한 스크립트들은 독립적으로 클라이언트에 전송되고 하이드레이션되며

그외의 나머지 페이지들은 정적 html로 렌더링되게 됩니다.
